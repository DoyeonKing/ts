## 项目简介

**项目名称**：餐剧盒-基于AI智能推荐的剧场文化社交平台

**项目类型**：大学生创新创业训练计划项目

**核心目标**：构建剧场文化社交平台，提供AI智能推荐服务

### 问题背景
- 剧场信息分散
- 用户互动不足
- 个性化推荐缺失

### 核心功能
- 知识图谱（剧目、演员、术语等）
- AI智能推荐
- 社交互动平台
- 票务服务

---

## 技术架构

### 前端
- **小程序框架**：微信小程序 / uni-app
- **UI框架**：原生组件 + 自定义组件库

### 后端
- **框架**：Spring Boot / FastAPI
- **API**：RESTful API

### 数据库
- **PostgreSQL**：业务数据存储
- **Neo4j**：图数据库（知识图谱）
- **Redis**：缓存
- **Elasticsearch**：搜索服务

### AI服务
- **大模型API**：豆包/文心一言/通义千问

## 答辩要点

### 技术选型理由
- **小程序**：用户使用便捷，无需下载安装
- **Neo4j**：适合知识图谱场景，支持复杂图查询
- **AI大模型API**：无需训练，成本可控，快速上线

### 项目创新点
1. 知识图谱 + AI推荐结合
2. 剧场文化垂直领域深耕
3. 一站式服务平台（知识+社交+票务）

### 项目价值
- 解决剧场信息分散问题
- 提升用户观剧体验
- 促进剧场文化传播
