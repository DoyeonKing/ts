# AI文创生成技术方案

## 核心回答（30秒）

AI文创生成功能使用支持图像生成的大模型API（豆包/文心一言），通过Prompt工程生成个性化的文创产品设计（海报、周边、纪念品等）。

**功能**：用户基于剧目、演员或观剧体验，让AI生成个性化文创设计。

**当前状态**：✅ 方案设计完成 ✅ 前端完成 ⚠️ 后端API实现中

---

## 技术实现

### 实现流程
1. **用户输入**：选择文创类型（海报/周边/纪念品/明信片/手账素材），输入剧目、演员、风格等
2. **构建Prompt**：后端根据输入构建结构化提示词
3. **调用API**：豆包/文心一言/通义千问（支持图像生成）
4. **结果处理**：接收图像，可选后处理（尺寸、水印），返回前端展示

### Prompt模板示例
```
基于以下信息，生成一个文创设计：
剧目：《雷雨》
演员：XXX
文创类型：海报
设计风格：现代简约
设计要求：包含剧目名称和主要元素，色调深色系，尺寸1920x1080
请生成符合要求的文创设计图。
```

### 支持的文创类型
- 海报（剧目/演员海报）
- 周边商品（T恤、帆布袋、钥匙扣等）
- 纪念品、明信片、手账素材

### API调用流程
用户输入 → 构建Prompt → 调用图像生成API → 接收图像 → 后处理（可选）→ 返回前端

## 常见问题简答

**Q: 质量如何保证？**  
设计详细Prompt模板；支持多次生成；结合剧目/演员真实信息；可引入人工审核。

**Q: 成本如何控制？**  
限制每日生成次数；选用成本较低的API；缓存热门设计；可设质量等级（高质量消耗积分）。

**Q: 可商用吗？**  
明确告知版权与使用限制；建议仅个人使用；商用需剧目授权；可与剧院合作提供官方授权服务。

**Q: 何时实现？**  
下一步：集成图像生成API → 实现Prompt构建与调用 → 图像处理与存储 → 前后端联调。
